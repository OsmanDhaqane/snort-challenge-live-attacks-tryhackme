# Snort Challenge - Live Attacks

This repository contains my walkthrough for the **Snort Challenge: Live Attacks** room on TryHackMe.

The task focused on using **Snort** to observe live malicious traffic, identify suspicious activity, and create **IPS rules** to block active attacks.

## What I worked on
- Capturing and reviewing traffic with Snort
- Identifying brute-force activity
- Detecting reverse-shell traffic
- Writing and testing custom Snort rules
- Running Snort in IPS mode to stop live attacks
